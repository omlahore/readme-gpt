README-GPT

⚡ Powered by Groq

This extension uses Groq's ultra-fast LLM to analyze GitHub repositories and provide intelligent explanations.

✅ Ready to analyze repositories!

How to use:

  1. Navigate to any GitHub repository
  2. Look for the "Explain this repo" button near the Star button
  3. Click it to get an AI-powered analysis

Features:

  • Repository overview and purpose
  • Key features and capabilities
  • Technology stack analysis
  • Project architecture insights
  • Getting started guide
  • Use cases and applications
  • Contributing guidelines

Supported file types:

README.md, package.json, requirements.txt, Cargo.toml, go.mod, Dockerfile, and 10+ more configuration files.

Note: The extension works with public GitHub repositories and uses Groq's ultra-fast LLM for analysis.
